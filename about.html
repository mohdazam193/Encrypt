About This Application

This application demonstrates a secure web architecture using AWS.

  • The frontend is hosted on an Amazon EC2 instance with an Nginx web server.
  • All cryptographic operations are handled by a serverless AWS Lambda function written in Python.
  • An Amazon API Gateway provides the secure endpoint for the frontend to communicate with the Lambda function.
  • The AES-256 encryption key is securely stored in AWS Secrets Manager, never exposed to the client or hardcoded.
